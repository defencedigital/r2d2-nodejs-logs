## R2D2 nodejs logs

This is a simple nodejs library used by r2d2 microservices to log messages to the console in unified json format.

## Usage

### package.json definition

```json
{
  "dependencies": {
    "@defencedigital/r2d2-nodejs-logs": "git+https://github.com/defencedigital/r2d2-nodejs-logs.git#v1.0.0"
  }
}
```

### Importing in typescript projects

```javascript
import Logger from '@defencedigital/r2d2-nodejs-logs';
```

### Importing in CommonJS projects

```javascript
const Logger = require('@defencedigital/r2d2-nodejs-logs');
```

### Importing in ES6 projects

```javascript
import Logger from '@defencedigital/r2d2-nodejs-logs';
```
